# 📝 IT23320932 – IT3040 Assignment 1

## 📌 Project Description
This project focuses on **Transliteration Accuracy Testing** for the "Chat Sinhala" feature of the **Pixelssuite** application. The primary objective is to evaluate how accurately the tool converts informal, chat-style Singlish inputs into Sinhala text through automated testing using Playwright.

*   **Target Application URL:** [https://www.pixelssuite.com/chat-translator](https://www.pixelssuite.com/chat-translator)
*   **GitHub Repository:** [https://github.com/kwg597/IT23320932_Playwright_Automation_Assignment.git](https://github.com/kwg597/IT23320932_Playwright_Automation_Assignment.git)

---

## 🎯 Assignment Scope
The scope of this automation project is strictly limited to the **Chat Sinhala** transliteration function.

### 🚫 Out of Scope:
*   Standard Sinhala transliteration testing.
*   Backend API, Performance, and Security testing.
*   Scalability and load testing.

📊 Test Case Summary
Total Scenarios: 50 Negative Test Cases.

Naming Convention: All Test IDs start with Neg_.

Coverage: Includes all 24 Singlish input types (Questions, Slang, Emojis, Mixed English/Singlish, etc.).

Logic: The script reads inputs from the Excel file, performs the transliteration on the live site, and compares the actual result with the expected Sinhala output.

Reporting: Actual output and Status (PASS/FAIL) are automatically updated in the IT23320932_Test cases.xlsx file.
