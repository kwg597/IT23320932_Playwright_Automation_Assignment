📝Project Description
This project focuses on Transliteration Accuracy Testing for the "Chat Sinhala" feature of the Pixelssuite application. The primary objective is to evaluate how accurately the tool converts informal, chat-style Singlish inputs into Sinhala text through automated testing using Playwright.

Target Application URL: https://www.pixelssuite.com/chat-translator  
GitHub Repository: https://github.com/kwg597/IT23320932_Playwright_Automation_Assignment.git

🎯 Assignment ScopeThe scope of this automation project is strictly limited to the Chat Sinhala transliteration function.  

🚫 Out of Scope:Standard Sinhala transliteration testing.  Backend API, Performance, and Security testing.  Scalability and load testing.

📂 Project StructureThe project follows the required submission structure:  PlaintextIT23320932/
├── IT23320932_test_automation.py  # Playwright automation script
├── IT23320932_Test cases.xlsx     # Excel report (50 negative cases)

🛠️ Technologies UsedPython: Core programming language.  Playwright: Browser automation framework for UI testing.  Pandas & OpenPyXL: Libraries for Excel data handling and reporting.

📊 Test Case Summary
Total Scenarios: 50 Negative Test Cases.  
Naming Convention: All IDs start with Neg_.  
Coverage: Includes all 24 Singlish input types (Questions, Slang, Emojis, etc.).  
Logic: Reads inputs from Excel, performs transliteration on the live site, and compares the actual result with the expected output.  
Reporting: Actual output and Status (PASS/FAIL) are automatically updated in the Excel file.
