# 📝 IT23320932 – IT3040 Assignment 1

## 📌 Project Description
This project focuses on **Transliteration Accuracy Testing** for the "Chat Sinhala" feature of the **Pixelssuite** application. The primary objective is to evaluate how accurately the tool converts informal, chat-style Singlish inputs into Sinhala text through automated testing using Playwright.

*   **Target Application URL:** https://www.pixelssuite.com/chat-translator
*   **GitHub Repository:** https://github.com/kwg597/IT23320932_Playwright_Automation_Assignment.git

---

## 🎯 Assignment Scope
The scope of this automation project is strictly limited to the **Chat Sinhala** transliteration function.

### 🚫 Out of Scope:
*   Standard Sinhala transliteration testing.
*   Backend API, Performance, and Security testing.
*   Scalability and load testing.

---

## 📂 Files Included
The project includes the following mandatory files organized within the registration number folder:

## 🛠️ Technologies Used
*   **Python**: Core programming language.
*   **Playwright**: Browser automation framework for UI testing.
*   **OpenPyXL**: Library for Excel data handling and reporting.

---

## ⚙️ Installation Steps
Before running this project, ensure you have Python installed. Then follow these steps:

1. **Open your terminal or VS Code.**
2. **Install the required Python libraries:**
   `pip install -r IT23320932_Requirements.txt`
3. **Install Playwright browsers:**
   `playwright install`

---

## 🚀 How to Run the Tests
To execute the automated tests and update the results in the Excel sheet, run the following command:

`python IT23320932_test_automation.py`

---

## 📊 Test Case Summary
*   **Total Scenarios**: 50 Negative Test Cases.
*   **Naming Convention**: All Test IDs begin with **Neg_**.
*   **Coverage**: Covers all 24 Singlish input types (Slang, Questions, Emojis, etc.).
*   **Execution**: The script automatically reads inputs, performs transliteration, and marks the status as **PASS** or **FAIL** in the Excel report.
