IT23320932 – IT3040 Assignment 1Project DescriptionThis project focuses on Transliteration Accuracy Testing for the "Chat Sinhala" feature of the Pixelssuite application. The primary objective is to evaluate how accurately the tool converts informal, chat-style Singlish inputs into Sinhala text through automated testing.  Target Application URL: https://www.pixelssuite.com/chat-translator  Assignment ScopeThe scope of this automation project is limited to the Chat Sinhala transliteration function.  Out of Scope:Standard Sinhala transliteration testing.  Backend API, Performance, and Security testing.  Scalability and load testing.  Project StructureThe project is implemented using Python and Playwright. Based on the submission requirements, the folder contains the following files:  PlaintextIT23320932/
├── IT23320932_test_automation.py  # Playwright automation script (Python)
├── IT23320932_Test cases.xlsx     # Excel report with 50 negative test cases
├── IT23320932_Requirements.txt    # List of Python dependencies
└── IT23320932_README.md           # Project documentation (this file)
Technologies UsedPython: Core programming language.  Playwright: Browser automation framework for UI testing.  Pandas & OpenPyXL: Libraries for Excel data handling and reporting.  Installation & SetupNavigate to the Project Folder:

How to Run the TestsTo run the automated transliteration tests and update the results in the Excel file, execute:  Bashpython IT23320932_test_automation.py
Test Case SummaryTotal Scenarios: 50 Negative Test Cases.  Naming Convention: All test case IDs start with Neg_.  Coverage: Includes all 24 Singlish input types (Questions, Commands, Slang, Emojis, Mixed English/Singlish, etc.).  Logic: The script reads inputs from the Excel file, performs the transliteration on the live site, and compares the actual result with the expected Sinhala output.  Reporting: The Actual output and Status (PASS/FAIL) are automatically updated in the IT23320932_Test cases.xlsx file.
