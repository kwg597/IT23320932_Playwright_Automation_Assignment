# 📝 IT23320932 – IT3040 Assignment 1

## 📌 Project Description
This project focuses on **Transliteration Accuracy Testing** for the "Chat Sinhala" feature of the **Pixelssuite** application. The primary objective is to evaluate how accurately the tool converts informal, chat-style Singlish inputs into Sinhala text through automated testing using Playwright.[cite: 1]

*   **Target Application URL:** [https://www.pixelssuite.com/chat-translator](https://www.pixelssuite.com/chat-translator)[cite: 1]
*   **GitHub Repository:** [https://github.com/kwg597/IT23320932_Playwright_Automation_Assignment.git](https://github.com/kwg597/IT23320932_Playwright_Automation_Assignment.git)[cite: 1]

---

## 🎯 Assignment Scope
The scope of this automation project is strictly limited to the **Chat Sinhala** transliteration function.[cite: 1]

### 🚫 Out of Scope:
*   Standard Sinhala transliteration testing.[cite: 1]
*   Backend API, Performance, and Security testing.[cite: 1]
*   Scalability and load testing.[cite: 1]
